# Autenticar Usuário

Cleyton acessa a página inicial do sistema ContabSys.

Ele verifica que o título da página exibido na aba do navegador e na tela é “Autenticação de Usuário”.

Ele verifica que existem campos para preenchimento de e-mail e senha, além do botão “Entrar”.

Cleyton informa seu e-mail e senha válidos.

Ele clica no botão “Entrar”.

O sistema valida se os campos foram preenchidos corretamente.

O sistema verifica se as credenciais informadas correspondem a um usuário ativo.

O sistema autentica o usuário e cria uma sessão válida.

O sistema redireciona Cleyton para a página inicial do sistema.

Cleyton verifica que seu nome e perfil de acesso são exibidos no topo da tela.
