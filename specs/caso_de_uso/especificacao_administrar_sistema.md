Administrar Sistema

Pré-condição: Usuário com perfil de Administrador.

O Administrador acessa o menu “Administração do Sistema”.
O sistema exibe as opções administrativas.
O Administrador seleciona a opção “Restaurar backup”.
O sistema solicita confirmação da operação.
O Administrador confirma a ação.
O sistema realiza a restauração dos dados.
O sistema reinicia os serviços necessários.
O sistema exibe uma mensagem informando que a restauração foi concluída com sucesso.
O Administrador verifica o funcionamento do sistema.

Resultado esperado: Sistema restaurado corretamente.
