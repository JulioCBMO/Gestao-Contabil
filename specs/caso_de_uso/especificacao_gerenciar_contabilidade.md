# Gerenciar Contabilidade

Cleyton acessa o menu “Contabilidade”.
O sistema exibe as opções relacionadas aos lançamentos contábeis.
Cleyton seleciona a opção “Novo lançamento contábil”.
O sistema exibe o formulário de lançamento.
Cleyton verifica que existem campos para data, valor, tipo de lançamento e plano de contas.
Cleyton preenche os campos com as informações contábeis.
Ele clica no botão “Registrar”.
O sistema valida os dados informados.
O sistema registra o lançamento contábil no banco de dados.
O sistema exibe uma mensagem confirmando o registro.
Cleyton verifica que o lançamento aparece na listagem contábil.
