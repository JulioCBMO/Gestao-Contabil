# Gerenciar Documentos

Cleyton acessa o menu “Documentos”.

O sistema exibe a lista de documentos já armazenados.

Cleyton seleciona a opção “Enviar documento”.

O sistema solicita a seleção de um arquivo.

Cleyton escolhe um arquivo em seu computador.

Ele clica no botão “Anexar”.

O sistema valida o tipo do arquivo.

O sistema armazena o documento.

O sistema exibe mensagem de sucesso.

Cleyton verifica que o documento aparece na lista.
