# Analisar Resultados

Cleyton acessa o menu “Relatórios”.

O sistema exibe as opções de relatórios disponíveis.

Cleyton seleciona o período e o tipo de relatório.

Ele clica no botão “Gerar relatório”.

O sistema processa os dados solicitados.

O sistema exibe o relatório na tela.

Cleyton analisa os resultados apresentados.

Cleyton seleciona a opção “Exportar em PDF”.

O sistema gera o arquivo PDF.
