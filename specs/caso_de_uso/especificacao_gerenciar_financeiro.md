# Gerenciar Financeiro

Carlos acessa o menu “Financeiro”.

O sistema exibe as opções de contas a pagar e a receber.

Carlos seleciona a opção “Gerar boletos”.

O sistema exibe o formulário de geração de boletos.

Carlos informa os dados financeiros necessários.

Ele clica no botão “Gerar”.

O sistema valida os dados informados.

O sistema gera os boletos financeiros.

O sistema exibe uma mensagem de confirmação.

Carlos verifica que os boletos aparecem na listagem financeira.
