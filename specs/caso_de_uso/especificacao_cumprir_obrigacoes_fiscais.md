# Cumprir Obrigações Fiscais

Carlos acessa o menu “Obrigações Fiscais”.

O sistema exibe as opções de apuração e transmissão fiscal.

Cleyton seleciona a opção “Apurar tributos”.

O sistema processa automaticamente os dados contábeis.

O sistema gera as guias de FGTS, GPS e arquivos fiscais.

O sistema exibe os documentos gerados para conferência.

Cleyton valida os valores apresentados.

Cleyton realiza a transmissão das obrigações fiscais.

O sistema confirma a transmissão com sucesso.
