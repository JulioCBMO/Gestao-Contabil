Gerenciar Empresas

Pré-condição: O usuário está autenticado no sistema.

Cleyton acessa o menu “Empresas” no sistema.
O sistema exibe a lista de empresas já cadastradas.
Cleyton verifica que existe a opção “Cadastrar nova empresa”.
Ele seleciona essa opção.
O sistema exibe a página de cadastro de empresas.
Cleyton verifica que existem campos para CNPJ, razão social e data de abertura.
Cleyton preenche os campos com os dados da empresa.
Ele clica no botão “Salvar”.
O sistema valida o formato do CNPJ informado.
O sistema confirma que o CNPJ não está duplicado.
O sistema cadastra a empresa.
O sistema exibe uma mensagem informando que a empresa foi cadastrada com sucesso.
Cleyton verifica que a empresa aparece na lista de empresas cadastradas.

Resultado esperado: Empresa cadastrada e disponível para uso.
