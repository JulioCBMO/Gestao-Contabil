# Gerenciar Folha de Pagamento

Cleyton acessa o menu “Folha de Pagamento”.

O sistema solicita a seleção da empresa e da competência.

Cleyton seleciona a empresa desejada e o mês de referência.

O sistema exibe a lista de funcionários ativos.

Cleyton seleciona a opção “Processar folha”.

O sistema calcula salários, descontos legais e encargos.

O sistema gera automaticamente os contracheques.

O sistema salva os dados da folha processada.

O sistema exibe uma mensagem informando que a folha foi processada com sucesso.

Cleyton verifica os valores apresentados.
